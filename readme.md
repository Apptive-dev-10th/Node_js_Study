# APPTIVE Study Repository
2018년도 여름방학, APPTIVE 10기 개발팀 스터디 내용

## Rules
* 주차별 스터디 내용은 깃헙을 통해서 공유한다.
* 스터디 주차별 마다 과제가 나가며, 과제의 결과물들은 개개인의 레포를 통해서 공유하며, issue 형태로 등록을 한다.
* 과제확인이 완료된 경우에는 issue를 closed 한다
